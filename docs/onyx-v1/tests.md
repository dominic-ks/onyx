# Onyx V1 — Test Log

> **Instructions:** Add every test as a new entry using the template below. Never edit past entries — append only. Most recent tests at the bottom.

---

## Template

```
---

### T[N] — [Short Test Name]

**Date:** YYYY-MM-DD

**Goal:**
What was being tested and why.

**Setup:**
- Environment (tank, bathtub, outdoor, dry bench, etc.)
- Configuration (what was installed, what wasn't)
- Any relevant measurements or initial conditions

**Result:**
What actually happened. Be factual, not interpretive.

**Observations:**
- Notable details (sounds, leaks, behaviour, anomalies)
- Anything unexpected

**Conclusion:**
What does this result mean? What was confirmed or disproved?

**Next Step:**
What should happen next based on this result.
```
